

# ZX-RTT-SDK

  [English Version](./README.md)  

ZX-RTT-SDK 包含了屏幕驱动、触摸功能和 UI 框架等，方便用户快速开发启明智显开发板。

已支持的开发板:

- ZX7D00M3R8-V10
- ZX7D00M4R64-V10
- ZX3D95CM20S-V11


支持的UI驱动, 默认为 LVGL-8.3.10

- LVGL-8.3.10
